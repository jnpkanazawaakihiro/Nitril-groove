# ニトリルグローブ販売LP（KW ホワイト & ダリア）

株式会社MEDI ALPHA PLUS が運営する、医療・検査関連機関向けニトリルグローブ販売LPです。

## プロジェクト概要

### 目的
医療機関・検査関連機関・関連法人向けに、ニトリルグローブのお見積CV（コンバージョン）を最大化するためのランディングページです。

### ターゲット
- 医療従事者
- 医療機関
- 検査関連機関
- 介護・福祉施設
- 食品関連事業者
- その他関連法人

### 主な特徴
- 薬機法・景表法・医療広告ガイドラインに配慮した表現
- BtoB向けの信頼感のあるデザイン
- 見積依頼・問い合わせに最適化されたCTA配置
- レスポンシブ対応（PC・タブレット・スマートフォン）
- 編集しやすいHTML構造

## 編集ガイド

### 1. テキスト内容の変更

#### ファーストビューのキャッチコピー
`index.html` の以下の部分を編集してください。

```html
<h1 class="text-3xl md:text-5xl font-bold text-gray-900 mb-6">
    医療・検査関連の現場ニーズを踏まえてご案内する、<br class="hidden md:block">業務用ニトリルグローブ
</h1>
```

#### 商品説明
各セクションの `<p>` タグ内のテキストを編集することで、説明文を変更できます。

#### 価格表示
価格は以下の箇所で変更可能です。

```html
<p class="text-center text-3xl font-bold text-blue-900">
    定価:1箱 1,500円(税抜)
</p>
```

## 法令対応について

このLPは以下の法令・ガイドラインに配慮して作成されています。

### 配慮した法令
- **景品表示法**（優良誤認・有利誤認の防止）
- **薬機法**（医療機器の広告規制）
- **医療広告ガイドライン**（比較優良広告・体験談の禁止）

### 表現上の注意点
- 「愛用」「No.1」「最高品質」等の最上級表現を避けています
- 「安心」「安全」等の保証表現を避けています
- 実績表現は具体的根拠がある場合のみ使用
- 一般個人への販売を制限することで、供給優先順位を明示

### 今後の注意事項
- 文言変更時は、断定表現や保証表現に注意してください
- 導入実績を追加する場合は、掲載許諾と証拠資料を確保してください

## デプロイ方法（GitHub Pages + お名前.com）

### 公開URL
- **本番サイト**: https://medialphaplus.shop
- **GitHub Pages**: https://[ユーザー名].github.io/[リポジトリ名]/

### デプロイ手順

#### 1. GitHubリポジトリの作成
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/[ユーザー名]/[リポジトリ名].git
git push -u origin main
```

#### 2. GitHub Pages設定
1. GitHubリポジトリの「Settings」→「Pages」
2. Source: `main` ブランチ、`/(root)` を選択
3. 「Save」をクリック

#### 3. お名前.comでDNS設定
以下のレコードを追加：

**Aレコード（4つ）**
```
ホスト名: @
TYPE: A
VALUE: 185.199.108.153
VALUE: 185.199.109.153
VALUE: 185.199.110.153
VALUE: 185.199.111.153
```

**CNAMEレコード**
```
ホスト名: www
TYPE: CNAME
VALUE: [GitHubユーザー名].github.io
```

#### 4. GitHubでカスタムドメイン設定
1. GitHub Settings → Pages → Custom domain
2. `medialphaplus.shop` を入力
3. 「Enforce HTTPS」にチェック（DNS反映後）

#### 5. DNS反映待ち
- DNS反映: 24〜48時間
- HTTPS証明書発行: 数分〜数時間
